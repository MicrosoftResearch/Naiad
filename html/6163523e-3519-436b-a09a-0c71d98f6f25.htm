Version 0.3Naiad Help

[This is preliminary documentation and is subject to change.]

The first release of Naiad with support for running computations on Microsoft Azure HDInsight and Hadoop 2.0 (YARN), version 0.3 was released alongside binary packages on NuGet.

Version 0.3 was released on April 4th 2014.

Changes in This Release

  • Added the ClusterSubmission solution, which contains support for launching Naiad programs in a variety of distributed settings, including:

    • Microsoft Azure HDInsight 3.0.
    • Hadoop 2.0 (YARN) on Windows.
  • The AzureSupport framework makes it easier to use Naiad programs with Microsoft Azure Storage Blobs and Tables.
