Examples

            To use this constructor, the programmer must pass a vertex factory, which is a function from an integer ID and stage to the vertex type (TVertex). The factory
            arguments must be passed through to the Vertex TTime (Int32, Stage TTime ) constructor. For example:

class MyVertex : Vertex<TTime> where TTime : Time<TTime>
{
    public MyVertex(int id, Stage<TTime> stage, ...)
        : base(id, stage)
    {
        /* Other initialization. */
    }
}

var stage = new Stage<TTime, MyVertex>(context, (i, s) => new MyVertex(i, s, ...), "MyStage");
