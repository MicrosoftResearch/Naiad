Naiad is an open source data-parallel dataflow system designed to provide low latency and high throughput for sophisticated data processing tasks. Naiad exposes low-level primitives for dataflow construction and execution, as well as several libraries wrapping standard patterns in higher level constructs. Our intent is that users will mostly use common high-level patterns, but have access to low-level primitives as needed, which can be wrapped and shared as new high-level patterns.

Getting Started

You will want a recent version of Naiad. Naiad is available both as binaries on NuGet and as open source on GitHub.
